About Dataset
Fictional E-Commerce Sales Data
Description
This dataset, titled 'Fictional E-Commerce Sales Data', consists of 10,000 rows of simulated sales data for a small e-commerce website. It was generated using synthetic data generation techniques and does not contain any real user information.

Fields in the Dataset
Order ID: A unique identifier for each order.
Product ID: A unique identifier for each product.
Product Category: The category to which the product belongs.
Quantity: The number of products ordered.
Unit Price: Price per unit of the product.
Total Price: Total price for the product(s) in the order (Quantity x Unit Price).
Order Date: The date when the order was placed.
Customer ID: A unique identifier for each customer.
Payment Type: Mode of payment (e.g., Credit Card, PayPal, etc.).
Order Status: Status of the order (e.g., Completed, Pending, Cancelled)
